# סקר הוועדה לקידום נשים בספורט

חבילה מוכנה לפרסום ב-GitHub Pages.

## מה יש כאן

- `index.html` - קובץ האתר הראשי. GitHub Pages יפתח אותו אוטומטית.
- `.nojekyll` - קובץ טכני שמונע עיבוד מיותר של GitHub Pages.
- `README.md` - קובץ הסבר בלבד.

## איך מעלים ל-GitHub Pages

1. פתחי Repository חדש ב-GitHub.
2. העלי את כל הקבצים שבתיקייה הזו לשורש ה-Repository, לא לתיקיית משנה.
3. היכנסי ל-Settings.
4. בתפריט הצדדי בחרי Pages.
5. תחת Build and deployment בחרי:
   - Source: Deploy from a branch
   - Branch: main
   - Folder: /root
6. לחצי Save.

לאחר כמה דקות יופיע קישור ציבורי בראש עמוד Pages.

## עדכון האתר בהמשך

כדי לעדכן את האתר, החליפי את הקובץ `index.html` בגרסה חדשה ושמרי את השינוי ב-GitHub.
